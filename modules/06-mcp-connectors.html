Model Context Protocol

An open standard where a host connects to servers that expose capability in a uniform way.

MCP standardises the envelope around tool calls, resources and prompt templates so an agent can discover what a server offers.

Connectors in practice

DiscoverThe connector advertises available tools, resources and schemas.
CallThe agent sends a structured request through the protocol.
ReturnThe connector returns results that become new context for the model.
Connectors are where a chatbot becomes a colleague that does things.

Same security questions as any integration: credentials, scope and trust.

End on the capability jump, but plant the risk seed: every connector is also an attack surface. Direct hand-off to Module 7.
